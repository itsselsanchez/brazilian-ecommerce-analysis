# 🛍️ Brazilian E-Commerce Analysis

E-commerce has become an ordinary part of daily life, but the systems behind it are anything but simple. The data generated by these platforms reflects a combination of customer behavior, operational constraints, and business outcomes that are rarely visible all at once.

Using the Olist Brazilian e-commerce dataset, this project explores the patterns, relationships, and tradeoffs that emerge across a large online marketplace through data preparation, exploratory analysis, feature engineering, and modeling.

🚧 **This project is currently in active development.**
